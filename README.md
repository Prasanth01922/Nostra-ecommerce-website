# Nostra – Fashion E-commerce Website

Nostra is a modern and responsive fashion e-commerce website created using **HTML5** and **CSS3**.
The project focuses on building a clean, elegant, and user-friendly shopping interface with responsive layouts and modern UI design principles.

The website includes multiple pages such as Home, Collections, and Contact Us, providing a complete front-end e-commerce experience.

---

## ✨ Features

* Fully Responsive Design
* Modern Fashion UI
* Sticky Navigation Bar
* Hero Section with CTA Buttons
* Product Showcase Cards
* New Arrivals Section
* Most Wanted Products Section
* CSS-only Mobile Navigation Menu
* Collections Filtering using CSS
* Contact Form UI
* FAQ Section
* Newsletter Subscription Section
* Smooth Hover Effects & Animations
